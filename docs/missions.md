# 미션 상세 명세

## 1. 미션 운영 원칙

- NEWSTART 8개 테마로 구성한다.
- 각 테마에는 2개의 미션이 있다.
- 각 테마에서 1개 이상의 미션을 성공하면 해당 테마를 클리어한다.
- 같은 테마의 2개 미션을 모두 성공하면 두 미션 점수를 모두 획득한다.
- 8개 테마를 모두 클리어하면 NEWSTART 완주 보너스 100점을 부여한다.
- 16개 전체 미션을 모두 완료하면 올클리어 보너스 200점을 부여한다.
- 일반 미션은 공통 QR 접속 후 미션 코드 입력 방식으로 접근한다.
- 히든 QR과 홍명기홀 최종 인증 QR만 별도 QR을 사용한다.

## 2. 미션 점수 기준

| 유형 | 점수 | 설명 |
|---|---:|---|
| 쉬운 미션 | 30 | 퀴즈, 간단 입력형 |
| 일반 미션 | 50 | 사진, 탐색, 이동, 작성형 |
| 도전 미션 | 80 | 스태프 판정, 현장 수행, 게임형 |

## 4. 테마별 상세 명세

## N. Nutrition 영양

### NUT-30: 영양 관련 QR 퀴즈

- 점수: 30점
- 인증 유형: quiz
- 완료 조건: 영양과 건강한 식습관 관련 퀴즈를 절반 이상 맞히면 성공
- 처리 방식: 자동 채점
- 테마 클리어: Nutrition

### NUT-50: 생명의 식탁을 완성하라

- 점수: 50점
- 인증 유형: text
- 완료 조건: 건강한 식단 카드 5개를 입력하면 성공. 오답 카드 포함 시 실패.
- 처리 방식: 정답 카드 리스트 기반 자동 판정 또는 관리자 검토

정답 카드 예시:

- 현미
- 채소
- 과일
- 견과류
- 물
- 달걀
- 닭가슴살
- 생선
- 두부
- 콩

오답 카드 예시:

- 튀긴치킨
- 탄산음료
- 냉동 패스트푸드
- 마라탕
- 컵라면
- 에너지드링크

운영 연출:

- 오답 카드는 “유혹 카드”로 표현한다.

## E. Exercise 운동

### EXE-50: 삼육대 무지개 루트

- 점수: 50점
- 인증 유형: photo
- 완료 조건: 캠퍼스에서 무지개 7가지 색을 찾아 사진 제출
- 처리 방식: 제출 즉시 임시 승인, 관리자 사후 검토 가능

색깔별 포즈:

| 색 | 포즈 |
|---|---|
| 빨강 | 열정 포즈 |
| 주황 | 점프 포즈 |
| 노랑 | 햇살 미소 |
| 초록 | 나무 포즈 |
| 파랑 | 파도 포즈 |
| 남색 | 기도 포즈 |
| 보라 | 왕관 포즈 |

인정 기준:

- 실제 현장에서 볼 수 있는 색만 인정한다.
- 사람 옷, 건물, 표지판, 꽃, 하늘, 조형물 등은 인정 가능하다.
- 디지털 화면이나 전자기기 색상은 미인정한다.
- 무지개 그림 하나는 한 가지 색으로만 인정한다.

### EXE-80: 네버스탑 1000보

- 점수: 80점
- 인증 유형: screenshot
- 완료 조건: 시작 시점과 종료 시점의 걸음 수 차이가 1,000보 이상
- 제출 자료: 시작 스크린샷, 종료 스크린샷
- 처리 방식: 관리자 검토 또는 임시 승인

## W. Water 물

### WTR-30: 생수의 근원을 찾아라

- 점수: 30점
- 인증 유형: quiz
- 완료 조건: 물 관련 퀴즈 절반 이상 정답
- 처리 방식: 자동 채점

### WTR-80: 하루 2리터 챌린지

- 점수: 80점
- 인증 유형: staff
- 스태프 필요: 예
- 진행 방식: 주어진 통에 물을 채워 2L를 맞춘다.
- 완료 조건: 연습 1회, 도전 3회 중 한 번이라도 2L 기준 100mL 내외이면 성공
- 처리 방식: 스태프가 관리자 화면에서 성공/실패 처리

## S. Sunshine 햇빛

### SUN-50: 빛을 찾아서

- 점수: 50점
- 인증 유형: photo
- 완료 조건: 햇빛이 잘 드는 캠퍼스 장소 3곳에서 사진 촬영
- 세 사진의 장소는 달라야 한다.
- 처리 방식: 제출 즉시 임시 승인, 관리자 사후 검토 가능

사진 콘셉트:

| 사진 | 콘셉트 |
|---|---|
| 1장 | 가장 밝은 미소 |
| 2장 | 햇빛을 받는 영웅 포즈 |
| 3장 | 하나님이 주신 빛을 표현하는 포즈 |

### SUN-80: 빛의 말씀을 찾아라

- 점수: 80점
- 인증 유형: text
- 완료 조건: 성경구절과 장절을 입력하면 성공
- 처리 방식: 정답 입력 자동 판정 또는 관리자 검토
- 운영 연출: 신학관 1층, 2층 복도에 성경구절 조각 카드 배치. 일부 가짜 조각 포함.

## T. Temperance 절제

### TMP-50: 절제의 3분

- 점수: 50점
- 인증 유형: video_or_text
- 완료 조건: 팀원 전원이 3분 동안 한 자리에 앉아 움직이지 않고 조용히 있는 모습을 인증
- 권장 처리: 영상 업로드는 부담이 크므로, 앱 내에서는 텍스트/외부 제출 안내 또는 시작/종료 사진 대체 가능
- 운영명: 절제의 3분: 웃참 챌린지

### TMP-80: 고통도 이겨내는 거야

- 점수: 80점
- 인증 유형: staff
- 스태프 필요: 예
- 진행 방식: 지압판 위에서 파티피리를 물고 줄넘기 10회
- 완료 조건: 팀 전원이 참여하되 참여자 과반수가 통과하면 성공. 소리를 내면 해당 인원 실패.
- 처리 방식: 스태프가 관리자 화면에서 성공/실패 처리

## A. Air 공기

### AIR-30: Air QR 퀴즈

- 점수: 30점
- 인증 유형: quiz
- 완료 조건: 공기 관련 퀴즈 절반 이상 정답
- 처리 방식: 자동 채점
- 퀴즈 유형: 수수께끼형 포함 가능

### AIR-80: 숨결로 날려버려

- 점수: 80점
- 인증 유형: staff
- 스태프 필요: 예
- 진행 방식: 얼굴에 붙은 포스트잇을 제한 시간 안에 떨어뜨린다.
- 완료 조건: 제한 시간 안에 떨어뜨리면 성공
- 처리 방식: 스태프가 관리자 화면에서 성공/실패 처리
- 판정 메모: 바람만 불어서 떨어뜨리는 것은 한계가 있으므로 어느 정도 얼굴을 흔드는 것은 인정

## R. Rest 휴식

### RST-50: 지친 자들아 티부스로 오라

- 점수: 50점
- 인증 유형: staff
- 스태프 필요: 예
- 진행 방식: 티부스에 방문하여 준비된 음료 또는 물을 마시며 쉰다.
- 완료 조건: 스태프 확인
- 처리 방식: 스태프가 관리자 화면에서 성공/실패 처리

### RST-55: 짐 내려놓기

- 점수: 50점
- 인증 유형: text
- 완료 조건: 익명 고민/기도 제목을 작성하고, 다른 팀의 기도 제목을 위해 1분간 기도
- 처리 방식: 제출 즉시 성공 처리 가능
- 검증 방식: 신뢰 기반 운영

작성 주제 예시:

1. 요즘 내가 내려놓고 싶은 걱정
2. 우리 팀이 함께 기도하고 싶은 제목
3. 오늘 하나님께 감사한 것
4. 친구에게 해주고 싶은 응원 한마디

## T. Trust 믿음

### TRS-50: 믿음의 한 컷

- 점수: 50점
- 인증 유형: photo
- 완료 조건: 오얏봉에서 팀원들이 서로 신뢰하는 장면을 창의적으로 연출해 사진 촬영
- 처리 방식: 제출 즉시 임시 승인, 관리자 사후 검토 가능
- 안전 기준: 위험한 자세, 위험한 장소, 업기, 높은 곳 올라가기 금지

랜덤 포즈 주제 예시:

- 보이지 않는 다리를 건너는 팀
- 한 명을 모두가 응원하는 장면
- 무너지는 친구를 붙잡는 장면
- 믿음으로 한 방향을 바라보는 장면
- 서로 손을 잡고 원을 만드는 장면
- 우리는 한 팀을 표현하는 장면

### TRS-80: 라이어 게임

- 점수: 80점
- 인증 유형: staff
- 스태프 필요: 예
- 진행 방식: 1명만 다른 제시어를 받고, 나머지는 같은 제시어를 받는다.
- 완료 조건: 각자 제시어 설명을 두 바퀴 진행한 뒤, 다른 제시어를 가진 사람을 찾아내면 성공
- 처리 방식: 스태프가 관리자 화면에서 성공/실패 처리

운영 규칙:

- 모든 제시어 아래에는 “본인은 진실된 제시어를 가지고 있습니다.” 문구를 넣는다.
- 게임 중 반응과 코멘트를 하지 않는다.
- 오로지 설명과 정답 지목 때만 표현할 수 있다.
